Häufige Fragen

Wie erstelle ich ein Profil?
Tippe oben rechts auf den Profil-Button → „Neues Profil" → Emoji und Name wählen → Fragen hinzufügen.


Wie plane ich ein Profil im Kalender?
Tippe oben links auf den Kalender-Button → Tag auswählen → Profil zuweisen. Das Profil aktiviert sich automatisch an diesem Tag.


Wie richte ich den iOS-Fokus-Filter ein?
iOS Einstellungen → Fokus → [Fokus auswählen] → App-Filter → Flowy → Modus wählen.


Werden meine Daten gespeichert oder übertragen?
Nein. Alle Daten bleiben ausschließlich lokal auf deinem Gerät. Flowy hat keinen Server.
